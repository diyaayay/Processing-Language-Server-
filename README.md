### Installation

- [Node.js](https://nodejs.org/) (version 18 or higher)
- [npm](https://www.npmjs.com/)

### Setup

```
git clone https://github.com/diyaayay/processing-language-server-extension.git
cd processing-language-server-extension/
npm install
npm run watch
```
